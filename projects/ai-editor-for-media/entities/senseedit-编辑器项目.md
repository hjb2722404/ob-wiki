# SenseEdit 项目

基于"主动感知"理念的 AI 原生智能编辑器，面向媒体编辑场景。

## 核心创新

**主动感知**——编辑器实时理解写作内容，主动提供写作建议，无需用户主动触发。

## 技术栈

- **核心**：TypeScript + ProseMirror Core
- **UI 适配**：Vue 3 / React 双适配
- **产品形态**：可嵌入宿主系统的编辑器组件

## 当前状态（2026年4月）

初始化阶段，仅有参赛文档，尚未开始编码。

## 目标规划

2026 年 4—7 月完成 MVP。
