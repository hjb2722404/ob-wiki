# 马基雅维利

## Basic Information
- Type: person
- Source: [[_raw/读书笔记/精品小说/冰与火之歌粉丝升级手册.md]]

## Description

尼科洛·马基雅维利（Niccolò Machiavelli，1469-1527）是意大利文艺复兴时期的著名政治思想家、历史学家和外交家，被公认为现代政治学的奠基人之一。他的代表性著作《君主论》（The Prince）系统地提出了现实主义的政治观点，对后世的政治理论和实践产生了深远影响，以至于形成了以他名字命名的政治哲学流派——[[concepts/权力的游戏|马基雅维利主义]]。

马基雅维利在政治哲学领域的核心贡献在于他对统治类型的区分。他将王国划分为世袭王国与新建立的王国：前者只需延续前任政策即可维持统治，而后者则需要面对更多来自被征服者的敌意与挑战。他指出权力的获取依赖于技巧和运气两个因素，而技巧并不等同于道德上的美德——这一观点颠覆了传统政治哲学将统治与道德美德等同的倾向。在《君主论》中，他甚至建议君主在必要时应当能够行不义之事，但必须谨慎使用残忍手段以避免树敌过多。

在统治者与民众的关系上，马基雅维利主张取悦平民比取悦贵族要容易得多，因为平民的唯一愿望就是不受压迫。这一关于[[concepts/权力的游戏|权力博弈]]的洞见，深刻影响了后世包括霍布斯在内的政治哲学家对国家与公民关系的研究。马基雅维利的现实主义政治观使其成为西方政治思想史上最具争议但也最具影响力的人物之一。
