# 《中论》

## Basic Information
- Type: product

## Description
《中论》（梵文：Mūlamadhyamakakārikā）是[[entities/龙树菩萨|龙树菩萨]]所著的中观学派根本论典，由[[entities/鸠摩罗什|鸠摩罗什]]大师于后秦时期译为汉文。《中论》与《百论》《十二门论》合称"三论"，是[[entities/三论宗|三论宗]]所依据的核心经典。隋朝[[entities/吉藏|吉藏]]大师（嘉祥大师）以三论为宗正式建立三论宗，以"破邪显正、二谛圆融"为宗趣。《中论》在[[concepts/中观学派|中观学派]]中的地位相当于"根本母论"，统领整个中观思想体系，其核心教义为"[[concepts/缘起性空|缘起性空]]"，以八不中道破斥诸法实有。
