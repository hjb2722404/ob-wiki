# 杨慎

## Basic Information
- Type: person
- Source: [[sources/最早来到山丹的两位古代帝王|最早来到山丹的两位古代帝王]]

## Description
杨慎（1488-1559），字用修，号升庵，是明代著名文学家，与[[entities/大禹|大禹]]传说中治水所立之碑有着重要的学术渊源。他曾将山丹发现的[[concepts/《大禹导弱水碑》|《大禹导弱水碑》]]蝌蚪文碑文译为汉文，共77字。该碑以[[concepts/蝌蚪文|蝌蚪文]]书写，相传为[[entities/大禹|大禹]]治水时所立，原迹早已湮没。南宋时此碑最早发现于湖南衡山岣嵝峰，宋明以降各地多有摹刻本。杨慎的翻译工作为今人理解这块千年古碑提供了珍贵的文字依据，在中国碑刻研究史上具有重要意义。杨慎学识渊博，著作丰富，与[[entities/林则徐|林则徐]]同为历史上的重要文化名人。
