# 《百字论》

## Basic Information
- Type: product
- Source: [[sources/百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记|百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记]]

## Description
《百字论》是[[entities/提婆菩萨|提婆菩萨]]所著的中观论典之一，全文仅有100字，是提婆三部论著中最为精炼的一部。该论以极简的文字浓缩了中观空性的核心教义，体现了提婆思想的高度凝练，言简义丰。在提婆的三部代表作中，《百字论》篇幅最短，类似于座右铭式的精要开示；与之相比，《百论》（100颂）结构清晰、逻辑严密，更适合作为[[concepts/中观学派|中观学派]]的入门读物；而《四百论》（400颂）则篇幅适中，论述较为详尽。三部论著共同构成了提婆中观思想的不同表达层次。
