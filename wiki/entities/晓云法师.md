# 晓云法师

## Basic Information
- Type: person
- Source: [[sources/痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记|痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记]]

## Description
晓云法师是现代华人佛教比丘尼与艺术教育家。她在文中被提及，作为说明《百喻经》教化理念与当代"觉之教育"思想相契合的代表人物。她提倡的"觉之教育"其本质是"邀请读者自我觉察，不强迫，只点灯"，强调以启发式、温和的方式引导个体走向觉醒，而非强制灌输。晓云法师将传统佛教的教化精神融入现代教育语境，主张教育者应像点灯人一样，为学习者提供自我觉察的契机与空间，使其在自愿的观照中开启修行与成长之路。
