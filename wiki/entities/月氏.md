# 月氏

## Basic Information
- Type: other
- Source: [[Clippings/最早来到山丹的两位古代帝王.md]]

## Description
月氏是古代活跃于[[concepts/河西走廊|河西走廊]]的游牧民族，与[[entities/乌孙|乌孙]]人共同生活在山丹大地。约前989年至前985年间[[entities/周穆王|周穆王]]西游河西时，月氏人和乌孙人是[[concepts/河西走廊|河西走廊]]的主要居民。方步和教授在《张掖史略》中论述，[[entities/焉支山|焉支山]]地区的游牧部落"也许是禺知（月氏）的一支，或者非禺知（月氏）的一支而是被禺知（月氏）统治的一支"。月氏在山丹地区演绎了众多历史故事，是山丹古代历史的重要组成部分。后来匈奴占领此地后，将[[entities/焉支山|焉支山]]名称从"焉居（支）"巧讹为"阏脂"或"胭脂"。
