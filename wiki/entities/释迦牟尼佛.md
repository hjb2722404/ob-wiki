# 释迦牟尼佛

## Basic Information
- Type: person
- Source: [[sources/阿含经核心教义讲解从原始佛教到大乘人间佛教的完整脉络-2026年06月04日-来自Get-笔记|阿含经核心教义讲解从原始佛教到大乘人间佛教的完整脉络-2026年06月04日-来自Get-笔记]]

## Description
释迦牟尼佛（乔达摩·悉达多）是阿含经中所记载的觉者，是原始佛教的创始人。据阿含经记载，释迦牟尼佛明确强调自己是以人身成就觉悟的凡人，亲口说出"我也在人数中"，坦然承认色身会经历成住坏空，会年老、背痛、疲倦，拒绝被神化崇拜，要求弟子[[concepts/人间佛教|依法不依人]]。

释迦牟尼佛出家前为迦毗罗卫国悉达多太子，曾跟随阿罗逻迦罗摩、郁陀迦罗摩子学习禅定，达到世间禅定最高境界后，发现仍无法断除轮回；又修了六年苦行，发现极端苦行同样不是正道，遂放弃苦行，接受牧羊女乳糜恢复体力，最终在菩提树下通过正念观察[[concepts/缘起法|十二因缘]]而证得觉悟。成道后他选择托钵乞食、游化人间，核心教义为[[concepts/四圣谛|四圣谛]]，每天为弟子说法、处理僧团事务、制定戒律。八十岁时在娑罗双树间入灭，临终遗言揭示诸行无常的真理。其主要弟子包括[[entities/舍利弗|舍利弗]]和[[entities/阿难|阿难]]等，[[entities/弥勒菩萨|弥勒菩萨]]则是其授记的将来佛。
