# 《百喻经》

## Basic Information
- Type: other
- Source: [[_raw/视频总结/《痴华鬘：千年佛教典籍百喻经的智慧与现代启示》-2026年06月04日-来自【Get 笔记】.md]]

## Description

《百喻经》是一部距今约一千六百年历史的佛教典籍，又名《痴华鬘》，意为愚痴故事串成的花环。该经由5世纪印度法师[[entities/僧伽斯那|僧伽斯那]]编撰，南齐永明十年（公元492年）由天竺高僧[[entities/求那毗地|求那毗地]]于建康正观寺汉译完成。全经包含一百则愚人故事，每则故事后附佛法开示，采用类型化扁平化叙述与[[concepts/遮诠|遮诠]]诠释方式，以应机说法的姿态揭示人类共通的[[concepts/无明|无明]]。一百则故事对应[[concepts/四圣谛|四圣谛]]的结构：展示[[concepts/三毒|三毒]]之苦，说明苦的集起（渴爱），宣说苦可息灭，并给出对治的修行路径，构成了完整的心灵地图。1914年[[entities/鲁迅|鲁迅]]曾出资金陵刻经处刊刻《百喻经》，推广此典籍。该经对东亚佛教文化与世俗文学均产生深远影响，是[[concepts/文字般若|文字般若]]的典型范例。
