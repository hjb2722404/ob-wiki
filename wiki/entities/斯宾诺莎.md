# 斯宾诺莎

## Basic Information
- Type: 人物（哲学家）
- Source: [[sources/“否定”的意义——研读黑格尔精神现象学的一点体会_叶秀山]]

## Description

斯宾诺莎（Baruch Spinoza，1632-1677）是荷兰籍犹太裔哲学家，被认为是现代西方哲学史上最具影响力的理性主义者之一。他以泛神论思想和几何学方法论著称，其哲学体系对后世黑格尔等人的发展产生了深远影响。

斯宾诺莎在哲学史上提出了一个著名命题：**"一切规定都是否定"**（Omnis determinatio est negatio）。这一原则指出，事物之所以具有特定的规定性，并非因为它们拥有某种肯定的内容，而是因为它们否定了其他可能性。例如，一张桌子之所以是桌子，正是因为它"不是"板凳。

在叶秀山研读黑格尔《精神现象学》的文章中，斯宾诺莎的这一命题被引申来解释[[concepts/经验世界|经验世界]]何以呈现"多姿多彩、万紫千红"的丰富性。正是通过[[concepts/否定|否定]]的作用，不同事物得以相互区分，世界才显得色彩斑斓。
