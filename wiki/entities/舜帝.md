# 舜帝

## Basic Information
- Type: person
- Source: [[Clippings/最早来到山丹的两位古代帝王.md]]

## Description
舜帝是上古时期的帝王，华夏大地洪水泛滥时，命大禹接替其父鲧治水。舜帝作为当时最高统治者，任命大禹治水的决定成就了大禹的历史功绩，也为大禹后来成为夏朝创立者奠定了基础。舜帝的知人善任体现了其卓越的政治智慧，其禅让之举更是被视为上古时期帝王美德的典范。这一任命不仅解决了当时的洪水灾难，更为华夏文明的发展奠定了重要基础。
