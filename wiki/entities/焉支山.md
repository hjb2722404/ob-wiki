# 焉支山

## Basic Information
- Type: 地理位置
- Source: [[sources/最早来到山丹的两位古代帝王|最早来到山丹的两位古代帝王]]

## Description

焉支山位于甘肃省山丹县东南部，是[[concepts/祁连山|祁连山]]的支脉，也是[[concepts/弱水|弱水]]发源地之一。原张掖师专[[entities/方步和|方步和]]教授考证，焉支山古名"焉居"或"支"，古时山麓宽阔草场游牧着一支氏族部落，以山为部落名，或以部落名山。[[entities/周穆王|周穆王]]《[[concepts/穆天子传|穆天子传]]》记载"己亥至于焉居、禺知之平"，此地即古代[[concepts/删丹|删丹]]地区。后来的匈奴将"焉支"巧讹为"阏脂"或"胭脂"，演变至今。焉支山地区是古代游牧民族活动的重要区域，[[entities/隋炀帝|隋炀帝]]曾在此山下召开万国博览会。[[concepts/山丹河|山丹河]]发源于焉支山及祁连山，是弱水上游。
