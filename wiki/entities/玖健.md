# 玖健

## Basic Information

- Type: person
- Source: [[_raw/读书笔记/精品小说/冰与火之歌.md]]

## Description

玖健是《冰与火之歌》系列中的重要角色，来自北境的绿先知（Greenseer），是森林之子（Children of the Forest）的后裔。他拥有三眼乌鸦所赋予的神秘能力，能够通过鱼梁木和旧神的力量看见过去与未来的幻象。玖健与其父亲霍通用·瑞德（Howland Reed）一同出现，他们家族世代守护着北境的古老秘密。作为绿先知，玖健展现了北境神秘力量的一面，他的角色深刻影响了故事的发展方向，特别是在引导琼恩·雪诺理解自身命运的过程中发挥了关键作用。玖健的智慧不仅体现在他对旧神的信仰上，更体现在他对阅读与人生价值的深刻理解之中。
