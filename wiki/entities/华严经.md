# 《华严经》

## Basic Information
- Type: product
- Source: [[sources/痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记|痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记]]

## Description
《华严经》是大乘佛教的核心经典之一，全称《大方广佛华严经》，在[[sources/痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记|《百喻经》智慧与现代启示]]的讲解中被频繁引用以印证因果积累的修行原则。文中在讲解"欲食半饼"喻时，引用《华严经》的核心教义——"一切诸佛都需要历经无量劫积累福慧资粮才能成正觉"——来说明修行没有捷径、每一步积累都算数的道理。该经与《心经》《维摩诘经》等经典并列为大乘佛教重要典籍，与[[sources/阿含经核心教义讲解从原始佛教到大乘人间佛教的完整脉络-2026年06月04日-来自Get-笔记|阿含经]]等早期经典共同构成大乘佛教的经典谱系。其教义深刻体现了[[concepts/缘起|缘起]]思想中关于因果聚合与修行次第的洞见，同时也对治修行中可能产生的[[concepts/慢心|慢心]]与速成心态，强调福慧双修的漫长积累过程。
