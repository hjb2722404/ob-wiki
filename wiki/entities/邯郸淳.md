# 邯郸淳

## Basic Information
- Type: person
- Source: [[sources/痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记|痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记]]

## Description
邯郸淳是三国时期人物，编撰了中国古代笑话集《笑林》。《笑林》收录了改编自《百喻经》中"治秃喻"故事的世俗笑话，但在改编过程中去除了原典的佛法开示部分，仅保留了笑点。这是《百喻经》故事传入中国后被世俗文学吸收的重要例证，展现了佛教典籍在中华文化中的本土化转变——从带有宗教教化功能的譬喻故事，演变为纯粹的世俗笑话。这一改编过程体现了佛教传播中[[concepts/yingjishuofa|应机说法]]的灵活性，以及[[concepts/wenzibanruo|文字般若]]在不同文化语境中的多样化呈现。邯郸淳的工作是将印度佛教譬喻文学中国化、世俗化的早期实践者之一。
