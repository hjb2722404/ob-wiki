# 费希特

## Basic Information

- Type: 人物（哲学家）
- Source: [["否定"的意义——研读黑格尔精神现象学的一点体会_叶秀山|"否定"的意义——研读黑格尔精神现象学的一点体会_叶秀山]]

## Description

约翰·戈特利布·费希特（Johann Gottlieb Fichte，1762-1814）是德国古典哲学的重要代表人物，被认为是为黑格尔的否定范畴奠定基础的关键哲学家。费希特试图通过将"否定"作为逻辑"环节"引入一元论本体论来克服[[concepts/康德]]的认识论二元论。他的核心创新在于将所有对象概念化为主体的否定，将一切存在概念化为思想的否定，这种逻辑策略类似于数学中的二进制计算。费希特确立了"第一原则"：从"A=A"中，通过中间的否定环节，涌现出"A≠非A"，从而使理性世界通过自我否定生成感性世界，同时保持二者的最终统一。作为连接[[concepts/康德]]与[[concepts/黑格尔]]哲学的关键人物，费希特的"大我"概念和[[concepts/自我意识]]理论对德国唯心主义的发展产生了深远影响，其[[concepts/知识论]]体系体现了[[concepts/理性一元论]]的基本立场。
