# 《隐藏的说服者》

## Basic Information
- Type: 其他（调查研究著作）
- Source: [[_raw/读书笔记/精品小说/《重访美丽新世界》笔记.md]]

## Description

《隐藏的说服者》（The Hidden Persuaders）是美国记者兼作家[[entities/万斯·帕卡德|万斯·帕卡德]]（Vance Packard）于1957年出版的调查研究著作。该书深入分析了广告商和营销人员如何利用[[concepts/潜意识诱服|潜意识诱服]]技术操控消费者心理和行为。[[entities/阿道斯·赫胥黎|阿道斯·赫胥黎]]在《[[sources/重访美丽新世界笔记|重访美丽新世界]]》中引用此书来说明现代[[concepts/广告宣传|广告宣传]]技术的精细化程度。书中描述了[[entities/海德|海德]]这样的「动机分析家」如何研究人性的弱点和缺点，调查人们无意识的欲望和恐惧，以便找出最佳方法去利用消费者的无知和荒唐，为雇主获取更多的金钱利益。帕卡德揭示了[[concepts/洗脑术|洗脑术]]式的[[concepts/笨蛋社会|笨蛋社会]][[concepts/广告宣传|广告宣传]]从简单的信息传递演变为潜意识操纵的全过程。
