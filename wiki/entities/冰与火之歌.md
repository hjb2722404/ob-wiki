# 冰与火之歌

## Basic Information
- Type: product
- Source: [[_raw/读书笔记/精品小说/冰与火之歌.md]]

## Description
《冰与火之歌》是一部由乔治·R·R·马丁创作的史诗奇幻小说系列，被视为现代奇幻文学的里程碑之作。系列以维斯特洛大陆为背景，讲述多个贵族家族之间为争夺铁王座而进行的权力斗争，同时伴随着异鬼威胁的北方边境危机。小说以其复杂的人物刻画、意外的剧情转折和深刻的道德灰色地带而广受赞誉。该系列已被改编为电视剧《权力的游戏》，在全球范围内获得了极高的知名度和影响力。
