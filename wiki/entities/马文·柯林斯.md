# 马文·柯林斯

## Basic Information

- Type: person
- Source: [[sources/马文·柯林斯的教育之道通往卓越教育的路径-马文·柯林斯-希维娅·塔玛金]]

## Description

马文·柯林斯（Marva Collins）是一位美国教育家，在芝加哥加菲尔德公园贫民区从事教育工作数十年。她并非教育专业出身，但在实践中形成了独特的教育理念。她在公共教育系统工作后，因对体制失望而创办了西区预备学校（[[entities/西区预备学校|西区预备学校]]），专门接收被其他学校放弃或被贴上"无学习能力"标签的学生。她的教育方法强调[[concepts/信念|信念]]、[[concepts/自尊培养|自尊]]、阅读和[[concepts/积极强化法|积极强化法]]，认为每个孩子都有学习的能力，只要给予适当的环境和动机。她的十个基本原则成为她教育实践的核心指导，包括相信每个学生、让每个学生每天都有收获、不放弃任何学生等。柯林斯拒绝了担任教育部长职位的机会，选择继续留在一线教学。她与[[entities/希维娅·塔玛金|希维娅·塔玛金]]合作，将其教育理念和方法整理成书，为更多教育工作者提供参考。

## Mentions in Source

- "一个优秀的教师，最需要的就是教育反思的能力。马文·柯林斯也不例外。如前所述，虽然她不是学教育专业的师范生，但是在教孩子的时候，她会思考自己的人生经历。"
- "她坚信，孩子们是想要学习，而且能够学习的，只要'给予他们恰当的环境、正确的动机和适合的学习材料，孩子们就能够展现出他们与生俱来的特长，脱颖而出'。"
- "马文·柯林斯还提出了十个基本原则——一是信念：相信每个学生能够攀上想象不到的高度；二是收获：让每个学生每一天收获满满，决不让学生失败；三是忠实：踏实做好每一件事情。"
