# 三论宗

## Basic Information
- Type: organization
- Source: [[sources/百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记|百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记]]

## Description
三论宗是由[[entities/吉藏|吉藏]]（嘉祥大师）创立的中国佛教宗派，以[[entities/鸠摩罗什|鸠摩罗什]]所译的《[[entities/中论|中论]]》、《[[entities/百论|百论]]》与《[[entities/十二门论|十二门论]]》三部论典为立宗根本，故名"三论"。该宗以"破邪显正、二谛圆融"为宗旨，强调通过破斥邪见来显示正见，主张正见是心的本来状态，破除覆盖本心的邪见则正见自然显现。

在法脉传承上，三论宗是[[concepts/中观学派|中观学派]]在中国汉地传承的重要载体。其传承谱系为：[[entities/龙树菩萨|龙树菩萨]]于佛灭五百年后出世开创中观，[[entities/提婆菩萨|提婆菩萨]]继承龙树并将中观逻辑推向顶峰，[[entities/鸠摩罗什|鸠摩罗什]]将三论译介至中土，最终由[[entities/吉藏|吉藏]]大师弘传并建立宗派。该法脉从印度传入中国，千年传承不绝，犹如"火光相传"。

三论宗的核心方法论是"破而不立"，即不建立任何正面主张，仅以逻辑破斥的方式扫荡一切执见，从而令实相自显。这一宗风深刻体现了般若中观的空性智慧，在中国佛教思想史上具有重要地位。
