# 大禹

## Basic Information
- Type: person
- Source: [[sources/最早来到山丹的两位古代帝王|最早来到山丹的两位古代帝王]]

## Description
大禹是中国古代传说中因治水而闻名天下的帝王，被认为是夏朝的创立者和首位君主。根据《尚书·禹贡》记载，禹导弱水至于合黎，余波入于流沙。山丹河发源于焉支山及祁连山，被认为是弱水之上游。据传大禹曾在石嘴山导弱水，这一传说从地形学角度有一定依据。山丹属古雍州，地势在九州中最高，有学者推测大禹治水遵循先高后低原则，自雍州之弱水始。《新语》记载大禹出于西羌，表明其为西部人，治理家乡水患合乎情理。1842年林则徐途经山丹时曾亲见《大禹导弱水碑》，碑文为蝌蚪文，共77字，碑已不知所踪。大禹被后世尊为"水利圣祖"，与尧、舜并称"三皇五帝"。

## Mentions in Source
- 第一位是公元前2000多年前因治水而造福万民的大禹。史传，上古时期，华夏大地洪水泛滥，舜帝命大禹接替父亲鲧治水。
- 《尚书·禹贡》载："（禹）导弱水，至于合黎，余波入于流沙。"弱水发源于山丹东南焉支山及祁连山，众多小河汇流于山丹城东郊，是为山丹河。
- 另，据西汉文学家陆贾《新语》云："大禹出于西羌"，如此说来，大禹是西部人，治理家乡水患，更属情理之中。
