# 山丹河

## Basic Information
- Type: 地理位置
- Source: [[Clippings/最早来到山丹的两位古代帝王.md]]

## Description
山丹河发源于山丹东南焉支山及祁连山，由众多小河汇流于山丹城东郊而成，是[[concepts/弱水|弱水]]之上游。据《尚书·禹贡》记载，大禹导弱水至于合黎，余波入于流沙，山丹河即为弱水上游的组成部分。据传[[entities/石嘴山|石嘴山]]为[[entities/大禹|大禹]]导弱水处，该地与野猫山两山对峙，中有山丹河西流。[[entities/大禹|大禹]]导弱水之后，为山丹河两岸人类繁衍生息创造了条件。从现有考古发掘可知，山丹境内四坝滩、壕北滩、山羊堡滩、花寨过会台、龙首山草场洼等靠近河流之处，均是山丹先民最早生产生活过的场所。
