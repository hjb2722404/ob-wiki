# 波吒厘城

## Basic Information
- Type: location
- Source: [[sources/百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记|百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记]]

## Description

波吒厘城（Pāṭaliputra）是古印度摩揭陀国的重要城市，亦即今印度比哈尔邦首府巴特那（Patna）的前身。该城在佛教史上具有重要地位，是大乘佛教中观学派传播与论辩活动的核心场所之一。提婆菩萨（[[entities/提婆菩萨|提婆菩萨]]）曾在此城与各派外道展开大规模论辩，逐一击败所有外道论师，声名远播。然而论辩的胜利也招致了外道弟子的怨恨，提婆最终在此城被外道弟子刺杀身亡。临终之际，提婆留下"诸法本空"的教诲，以自身的死亡示现无我证量，将中观破执的智慧从理论辩论推向生命的终极实践，使波吒厘城成为中观学派重要精神坐标之一。
