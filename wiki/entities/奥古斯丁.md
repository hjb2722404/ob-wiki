# 奥古斯丁

## Basic Information

- Type: person
- Source: [[sources/冰与火之歌粉丝升级手册]]

## Description

奥古斯丁（Saint Augustine of Hippo，354-430）是希波城的主教，早期基督教神学家、哲学家，对西方基督教神学和哲学产生深远影响。他与[[柏拉图]]的哲学传统有着深厚的联系，同时在神学问题上与后来的[[莱布尼茨]]等启蒙时期哲学家形成跨时代的对话。本书在讨论邪恶问题时引用奥古斯丁的核心观点：恶并不是独立存在的事物，恶只是善的缺失（恶是善的缺失状态）。他还提出著名的自由意志辩护理论：上帝让人类具有自由意志，即选择自己行为的能力；如果上帝阻止恶，也就同时剥夺了个人的自由意志。这一思想成为基督教神正论的重要基础，也是哲学史上讨论[[邪恶问题]]时无法绕过的经典论证。
