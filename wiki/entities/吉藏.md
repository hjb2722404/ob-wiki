# 吉藏

## Basic Information
- Type: person
- Source: [[sources/百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记|百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记]]

## Description
吉藏大师，又称嘉祥大师，是中国佛教[[entities/三论宗|三论宗]]的创立者。他以[[entities/鸠摩罗什|鸠摩罗什]]所译的《中论》《百论》《十二门论》三部论典为宗依，建立了以"[[concepts/破邪显正|破邪显正]]"与"[[concepts/二谛圆融|二谛圆融]]"为宗旨的三论宗体系。吉藏属于[[concepts/中观学派|中观学派]]在中国的重要传承者，其学说直接继承自印度大乘佛教的中观传统。在《百论》的注疏与弘传过程中，吉藏提出了一个核心理念："破邪即是显正"——他认为正见的障碍就是邪见，破除邪见后，心本来的清净正见自然显现，如同擦去镜子上的灰尘，镜子本来的光洁自然恢复，不需要在破邪之外额外建立新的正见。这一思想将[[concepts/破邪显正|破邪显正]]从一种修行方法提升为本体论层面的洞察，强调"破"与"显"的当下统一。
