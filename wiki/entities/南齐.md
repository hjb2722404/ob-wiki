# 南齐

## Basic Information
- Type: event
- Source: [[sources/痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记|痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记]]

## Description
南齐（479-502年）是中国南北朝时期南朝的重要政权之一，由萧道成所建，因皇室姓萧，故又称"萧齐"。南齐享国仅二十余年，但永明年间（483-493年）是其文化与佛教事业的重要时期。公元492年（永明十年），天竺高僧[[entities/求那毗地|求那毗地]]在[[entities/建康|建康]][[entities/正观寺|正观寺]]完成《百喻经》的汉文翻译工作，使这部源自古印度的千年佛典得以传入中国。南齐时期的佛经翻译事业繁荣，对后世中国佛教的发展影响深远。
