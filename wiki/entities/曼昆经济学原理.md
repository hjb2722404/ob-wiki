# 曼昆经济学原理

## Basic Information
- Type: product
- Source: [[_raw/读书笔记/经济理财/《曼昆经济学原理》（三）.md]]

## Description

曼昆经济学原理是由哈佛大学教授N. Gregory Mankiw编写的经典经济学入门教材，被全球众多高校广泛采用。该教材以清晰易懂的表达方式阐释基础经济学概念，成为经济学教育的标杆之作。《曼昆经济学原理（三）》作为读书笔记的第三卷，重点涵盖了市场均衡理论、弹性理论以及价格管制政策等微观经济学的核心内容。书中通过现实世界的案例和数学公式，系统介绍了需求价格弹性、供给价格弹性、均衡价格的形成机制，以及价格上限与价格下限对市场的影响。该教材为初学者提供了扎实的经济学分析基础。
