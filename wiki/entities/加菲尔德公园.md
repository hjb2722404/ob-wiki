# 加菲尔德公园

## Basic Information

- Type: location
- Source: [[sources/马文·柯林斯的教育之道通往卓越教育的路径-马文·柯林斯-希维娅·塔玛金]]

## Description

加菲尔德公园是位于美国芝加哥的一个贫困社区，属于该市教育资源匮乏、经济发展滞后的地区之一。这一社区在马文·柯林斯（Marva Collins）的早期教学生涯中占据重要位置——她正是在加菲尔德公园开始了对贫困家庭非裔美国人工人阶层子女的教育工作。这些孩子的家庭普遍缺乏教育资源和社会支持，面临着主流教育系统长期忽视的现实困境。

加菲尔德公园代表了柯林斯所面对的社会环境：一个被主流教育体制边缘化的贫困社区。尽管面临诸多挑战，柯林斯仍致力于通过广泛的知识传授——从数学词源到经典文学——为这些孩子打开一个更大的世界，让他们意识到自己也有渴求的东西，超越加菲尔德公园贫民窟的局限。
