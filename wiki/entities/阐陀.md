# 阐陀

## Description
阐陀 (Chandaka) 是[[entities/释迦牟尼佛|释迦牟尼佛]]的一位重要弟子，其生平事迹记载于阿含经等[[sources/阿含经核心教义讲解从原始佛教到大乘人间佛教的完整脉络-2026年06月04日-来自Get-笔记|阿含经典]]中。他在出家前原为[[entities/释迦牟尼佛|悉达多太子]]的御者（车匿），后随[[entities/释迦牟尼佛|佛陀]]出家为比丘。出家后的阐陀以"我慢极强"著称，是整个僧团中"最难调伏"的弟子，其自我执着之重在[[entities/释迦牟尼佛|佛陀]]诸大弟子中堪称第一。

佛陀在世时，阐陀虽为比丘，但其强烈的自我中心与不驯服的性格使僧团难以管教。佛陀入灭后，僧团为使其反省自身过失，对其实施了[[concepts/默摈法|默摈法]]（一种僧团共议的冷处理惩戒），通过集体沉默的方式令其直面自身问题。在这一强大的因缘促迫下，阐陀终于痛加忏悔，破除我执，最终证得[[concepts/阿罗汉|阿罗汉]]果位。

阐陀的故事在原文中被用来印证一个重要的修行观点：即便"我挡（慢）最重"的人，只要因缘具足，同样能够觉悟证果。这为[[concepts/人间佛教|人间佛教]]"烦恼重者亦可成佛"以及[[concepts/菩萨行|菩萨行]]"不舍一人"的精神提供了经典依据。其事迹常被引述为对照[[entities/舍利弗|舍利弗]]、[[entities/阿难|阿难]]等易于调伏之弟子的另一极端案例。
