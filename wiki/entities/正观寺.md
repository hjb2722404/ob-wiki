# 正观寺

## Basic Information
- Type: location
- Source: [[sources/痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记|痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记]]

## Description
正观寺是南齐时期位于[[entities/建康|建康]]（今江苏南京）的一座佛寺，建于南朝，是当时建康城内的重要佛教活动场所。公元492年（南齐永明年间），天竺高僧[[entities/求那毗地|求那毗地]]在此驻锡，完成了《百喻经》的汉文翻译工作，将这部以譬喻故事宣讲佛教教义的印度典籍系统地介绍到中国。正观寺因此成为《百喻经》传入中国的关键场所，也见证了南朝时期中印佛教文化交流的重要篇章。其寺名"正观"取自佛教"正观"（如实观照、正见正观）之义，体现了该寺以佛教正法修持为本的宗风。
