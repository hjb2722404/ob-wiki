# 方步和

## Basic Information
- Type: 人物
- Source: [[sources/最早来到山丹的两位古代帝王|最早来到山丹的两位古代帝王]]

## Description
方步和是原张掖师专教授，专注于张掖地区历史研究，著有《张掖史略》一书。在论述[[entities/焉支山|焉支山]]历史时，方步和教授考证了"焉居"与焉支山的关系，以及[[entities/月氏|月氏]]与当地部落的关系。他认为古时焉支山麓的宽阔草场游牧着一支氏族部落，以山为部落名叫做焉居，或者以部落名山叫做焉支山，而以部落名山的可能性更大。这支游牧部落可能是月氏的一支，或是被月氏统治的一支。方步和的考证为理解山丹古代历史提供了重要的学术支撑。
