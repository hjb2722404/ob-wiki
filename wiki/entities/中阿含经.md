# 中阿含经

## Basic Information
- Type: other（佛教经典文本）
- Source: [[sources/阿含经核心教义讲解从原始佛教到大乘人间佛教的完整脉络-2026年06月04日-来自Get-笔记|阿含经核心教义讲解从原始佛教到大乘人间佛教的完整脉络-2026年06月04日-来自Get-笔记]]

## Description
中阿含经是早期佛教[[concepts/四圣谛|四圣谛]]教义体系的核心经典之一，属于四部阿含（《杂阿含》《中阿含》《长阿含》《增一阿含》）中的重要一部。其风格偏向理性分析与法义抉择，篇幅介于[[entities/杂阿含经|杂阿含]]与[[entities/长阿含经|长阿含]]之间，特色在于层层辨析推论，详细解释正邪见解与修行道路。

该经收录了大量关于[[concepts/止观双运|止观双运]]、戒律、禅定与修行实践的具体教导，是面向理性思维学习者的详尽修行手册。经中包含[[entities/释迦牟尼佛|释迦牟尼佛]]明确驳斥宿命论的重要记载，指出现在的境遇是过去定业的果报，但未来属于不定业，取决于当下的选择；过去的恶业可以通过现在大量修善加以稀释，甚至实现"重报轻受"，强调[[concepts/业力|业力]]可由当下行为所改变，命运并非全然前定。
