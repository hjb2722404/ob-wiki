# 建康

## Description
建康是中国南朝时期的都城，位于今江苏省南京市，古称"建业"，是六朝（孙吴、东晋、宋、齐、梁、陈）政治、经济与文化的核心区域。公元492年（南齐永明十年），天竺高僧[[entities/求那毗地|求那毗地]]于建康的[[entities/正观寺|正观寺]]完成了佛教经典《百喻经》的汉文翻译工作，使这部以寓言形式宣扬佛理的典籍得以传入中土。建康在南朝时期汇聚了大量中外僧侣与译经高僧，不仅是汉传佛教的重要传播枢纽，也是当时东亚最具影响力的文化都市之一。
