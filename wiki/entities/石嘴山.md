# 石嘴山

## Basic Information
- Type: 地理实体（location）
- Source: [[sources/最早来到山丹的两位古代帝王|最早来到山丹的两位古代帝王]]

## Description

石嘴山位于山丹祁家店水库堤坝南端，据传为[[entities/大禹|大禹]]导弱水之处。该地与[[野猫山]]两山对峙，中有[[entities/山丹河|山丹河]]（弱水上游）西流。从地形学角度看，这一传说有一定依据。

山丹属古雍州，在九州中地势最高。根据[[治水]]先高后低的原则，[[entities/大禹|大禹]]治水自雍州之[[弱水]]、黑水始有一定合理性。石嘴山作为[[entities/大禹|大禹]]导弱水的传说地点，是山丹与古代帝王历史联系的重要地理标识，见证了大禹治水传说在山丹地区的流传。
