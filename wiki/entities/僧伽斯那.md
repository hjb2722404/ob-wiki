# 僧伽斯那

## Basic Information

## Description
僧伽斯那是5世纪古印度佛教法师，《百喻经》的编撰者。他从印度古代民间寓言集《五卷书》和《故事海选》中选取了一百则愚人故事，根据佛法需要对每则故事重新改编，并在每则后附佛法开示，兼具民间文学的生动和佛教典籍的深度。其生卒年月没有明确可考的记录。他在《百喻经》跋颂中提出"取走药，弃掉叶"的核心理念，确立了《百喻经》以故事疗愈内心的宗旨——故事的笑声是叶，对治贪瞋痴的智慧才是药。《百喻经》原名《痴华鬘》，意为愚痴故事串成的花。《百喻经》由南朝齐天竺法师[[entities/求那毗地|求那毗地]]于公元492年译介至中国。
