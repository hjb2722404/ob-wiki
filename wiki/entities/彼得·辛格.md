# 彼得·辛格

## Basic Information
- Type: 人物
- Source: [[sources/冰与火之歌粉丝升级手册]]

## Description

彼得·辛格（Peter Singer，1946年—）是澳大利亚哲学家，现任普林斯顿大学生物伦理学教授，是当代最具争议性和影响力的伦理学家之一。他因在[[concepts/功利主义|功利主义]]和[[concepts/动物权利|动物权利]]领域的开创性工作而闻名，其著作《动物解放》被认为是[[concepts/动物权利|动物权利]]运动的奠基之作。辛格的研究涵盖生命伦理、道德哲学和应用伦理学等领域，对现代伦理思考产生了深远影响。

辛格的一个著名而有争议的论断是：杀死一个严重残疾的婴儿不构成杀人罪。他基于[[concepts/功利主义|功利主义]]立场提出这一观点，认为应当考虑婴儿未来的痛苦和负担。这一论断引发了广泛伦理争议，涉及生命权、残疾权利和[[concepts/功利主义|功利主义]]计算等核心议题，与[[concepts/人格概念|人格概念]]的哲学探讨密切相关。
