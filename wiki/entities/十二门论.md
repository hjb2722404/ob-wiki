# 《十二门论》

## Basic Information
- Type: product
- Source: [[sources/百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记|百论全品深度解读]]

## Description
《十二门论》（梵文：Dvādaśadvāraśāstra）是[[entities/龙树菩萨|龙树菩萨]]所著的中观论典之一，由[[entities/鸠摩罗什|鸠摩罗什]]大师于后秦时期翻译为汉文，是中观法脉传入中土的重要译著之一。《十二门论》与[[entities/中论|《中论》]]、[[entities/三论宗|《百论》]]（[[entities/三论宗|《百论》]]为[[entities/三论宗|提婆菩萨]]所著）合称"三论"，共同构成[[entities/三论宗|三论宗]]的核心经典体系。全论以十二个门类（十二门）分别阐述中观空性思想，系统破除执见、显扬空义，是龙树中观体系的入门纲要之一，与篇幅浩瀚的[[entities/中论|《中论》]]相辅相成、互为表里。[[entities/吉藏|吉藏]]（嘉祥大师）以三论为宗正式开创[[entities/三论宗|三论宗]]，以"破邪显正、二谛圆融"为宗趣，《十二门论》在此宗派义理建构中具有不可替代的经典地位。
