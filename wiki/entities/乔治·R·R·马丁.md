# 乔治·R·R·马丁

## Basic Information

- Type: person
- Source: [[_raw/读书笔记/精品小说/冰与火之歌.md]]

## Description

乔治·R·R·马丁（George R.R. Martin，简称GRRM）是一位美国著名作家，以创作史诗奇幻小说闻名于世。他最著名的作品是《[[sources/冰与火之歌|冰与火之歌]]》系列小说，该系列是全球最畅销的史诗奇幻作品之一，包含《权力的游戏》、《列王的纷争》、《冰雨的盛宴》、《群鸦的盛宴》、《与龙共舞》等多部作品。

马丁以其对角色的复杂刻画而著称，他大胆拒绝遵循传统奇幻小说"主角不死"的规则，创造了一个充满政治阴谋、战争和超自然威胁的黑暗幻想世界。他通过写实的写作风格和对角色命运不可预测性的处理，重新定义了现代奇幻文学的叙事标准。

其创作的《[[concepts/权力的游戏|权力的游戏]]》被 HBO 改编为电视剧后，成为现象级文化作品，在全球范围内引发收视热潮。该剧继承了马丁作品中残酷而真实的叙事风格，将权力的博弈与人性的复杂展现得淋漓尽致。
