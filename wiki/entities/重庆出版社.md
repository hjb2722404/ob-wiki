# 重庆出版社

## Basic Information

- Type: organization
- Source: [[_raw/读书笔记/精品小说/冰与火之歌.md]]

## Description

重庆出版社是中国重庆市的一家综合性出版社，在出版领域有着重要影响力。该出版社负责《冰与火之歌》系列中文版的出版工作，将乔治·R·R·马丁的经典奇幻文学作品引入中国市场。通过出版马丁的系列作品，重庆出版社在中国奇幻文学读者群体中具有重要地位，为中国读者搭建了接触西方奇幻文学的重要桥梁。作为《冰与火之歌》中文版的出版方，重庆出版社在该作品的传播与推广过程中发挥了关键作用。
