# 大自在天

## Basic Information
- Type: other
- Source: [[sources/百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记|百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记]]

## Description
大自在天是印度外道所崇拜的最高神祇，又称 Maheśvara（摩醯首罗），被视为永恒的创造主与世界的主宰者。在印度宗教传统中，大自在天被认为是常住不灭、具足自在力的最高存在，外道依此建立神我论（Ātman论）的宗教体系。在《百论》记载的故事中，[[entities/提婆菩萨|提婆菩萨]]曾亲身前往大自在天神殿，以取出神像眼睛的果敢行动，对大自在天信仰提出两难破斥：若神有情识则属无常之物，若神无情识则无能力创造世界。此示现以实际行动揭露了外道神祇观念在理性分析下的根本矛盾，体现了中观学派破除外道执着的智慧。
