# 周穆王

## Basic Information
- Type: person（人物）
- Source: [[sources/最早来到山丹的两位古代帝王|最早来到山丹的两位古代帝王]]

## Description
周穆王（约前1026年—前922年），姓姬名满，是周武王的第四代孙，西周第五位天子，在位约55年。根据《竹书纪年》记载，周穆王曾于穆王十三年（前989年）、穆王十七年（前985年）两次西游，是有文字记载以来最早西行的中国帝王之一。周穆王自洛阳西巡，经河套进入[[concepts/河西走廊|河西走廊]]，经过[[concepts/删丹|删丹]]（今甘肃山丹）后继续西行，过黑河至居延海，在当地部落首领[[entities/柏夭|柏夭]]的向导下，抵达"羽琌之地"。其西行路线经过[[concepts/焉支山|焉支山]]得到学界普遍认可，当时生活在[[concepts/河西走廊|河西走廊]]的是[[entities/月氏|月氏]]人和[[entities/乌孙|乌孙]]人。《[[concepts/穆天子传|穆天子传]]》详细记载了这次西行旅程。
