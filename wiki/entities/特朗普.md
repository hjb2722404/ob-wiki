# 特朗普

## Basic Information
- Type: person
- Source: [[sources/曼昆经济学原理二|曼昆经济学原理二]]

## Description
特朗普（Donald Trump）是美国的第45任总统（2017-2021），在讨论国际贸易政策和经济理论时经常被提及。根据[[sources/曼昆经济学原理二|曼昆经济学原理二]]的记录，特朗普的支持者主要来自美国蓝领工人群体，这些工人因国际贸易和全球化而失去工作或收入下降。特朗普的贸易战政策反映了其试图将不具备比较优势的生产活动引回美国的意图，这与经济学中关于比较优势的理论形成鲜明对比。这一现象说明了经济理论如何产生现实的政治影响。
