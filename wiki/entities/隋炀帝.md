# 隋炀帝

## Basic Information
- Type: person
- Source: [[sources/最早来到山丹的两位古代帝王|最早来到山丹的两位古代帝王]]

## Description
隋炀帝是隋朝第二位皇帝，名杨广，庙号世祖。他曾西巡[[张掖]]，在[[焉支山]]下召开[[万国博览会]]、接见西域二十七国王公大臣。这一历史事件在山丹几乎人尽皆知，是山丹历史上最著名的帝王访问之一。然而，早在炀帝之前的两三千年，就已有[[大禹]]和[[周穆王]]两位帝王来到过山丹，因此隋炀帝并非最早来到山丹的帝王。隋炀帝的西巡展示了他对西域经营的政治抱负，体现了隋朝对丝绸之路的经营战略。
