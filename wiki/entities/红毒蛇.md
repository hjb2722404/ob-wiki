# 红毒蛇

## Basic Information
- Type: person
- Source: [[sources/冰与火之歌|冰与火之歌]]

## Description
红毒蛇是奥伯伦·马泰尔亲王的绰号，是多恩领地的著名人物，以其毒药知识和嗜酒成性的生活方式而闻名。他是一个复杂且具有争议的角色，在对女儿们的告诫中展现了对婚姻和女性自主的独特见解。他的名言体现了对女性独立和谨慎选择伴侣的重视。作为多恩的马泰尔家族成员，他展现了独特的人生哲学，认为世上快乐稀少，人们应自主选择生活方式，同时也强调谨慎选择伴侣的重要性，以免被"笨蛋或暴徒"缠上。
