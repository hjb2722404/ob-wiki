# 哈珀·李

## 基本信息

- **类型**: 人物
- **来源**: [[sources/邻家微光]]

## 描述

哈珀·李（Nelle Harper Lee，1926年4月28日—2016年2月19日），美国著名作家，以其代表作《杀死一只知更鸟》（To Kill a Mockingbird）闻名于世。该小说于1960年出版，翌年便荣获普利策小说奖。哈珀·李出生于美国阿拉巴马州门罗维尔，在当地度过了大部分童年时光。她与著名作家[[杜鲁门·卡波特|杜鲁门·卡波特]]（Truman Capote）是青梅竹马，两人曾共同度过童年时光。卡波特后来创作了《冷血》等重要作品。哈珀·李的写作生涯相对低调，除了《杀死一只知更鸟》外，她还出版了续篇《设立守望者》（Go Set a Watchman），但后者在文学价值和真实性方面存在争议。*(No source content available for this page)*
