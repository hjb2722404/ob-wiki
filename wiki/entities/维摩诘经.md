# 维摩诘经

## Basic Information
- Type: product
- Source: [[sources/痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记|痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记]]

## Description
《维摩诘经》（全称《维摩诘所说经》）是大乘佛教经典之一，以维摩诘居士与文殊菩萨、舍利弗等佛弟子展开辩驳的对话形式说教，呈现"在家居士"修行般若的精深境界。该经以"辩才无碍"著称，在文中被用作与[[entities/百喻经|《百喻经》]]语言风格的对比参照：《维摩诘经》采用"逼拶式"的辩锋语言，以层层逼问、锋芒直指见性；而《百喻经》则以浅白引人发笑的方式"以[[concepts/遮诠|遮诠]]见长"，属于[[concepts/文字般若|文字般若]]的灵活运用。两者代表了大乘佛教弘法语言的两种不同路径：锋利逼问与轻松寓教。前者以思辨的紧张感破执，后者以笑声的松动感入心。
