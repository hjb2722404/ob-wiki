# 乌孙

## Basic Information
- Type: 古代游牧民族
- Source: [[Clippings/最早来到山丹的两位古代帝王.md]]

## Description
乌孙是中国古代活跃于[[entities/焉支山|焉支山]]地区及[[concepts/河西走廊|河西走廊]]一带的游牧民族。据史料记载，在[[entities/周穆王|周穆王]]西巡期间（约前989年-前985年），乌孙人与[[entities/月氏|月氏人]]共同生活在山丹大地，是这一地区的主要居民。这两支剽悍的游牧民族在山丹大地上演绎了众多惊心动魄的历史故事，构成了山丹古代历史的重要组成部分。乌孙族以其勇猛善战著称，在古代丝绸之路上留下了深刻的历史印记。
