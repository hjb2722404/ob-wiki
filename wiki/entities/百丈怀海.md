# 百丈怀海

## Basic Information
- Type: person
- Source: [[sources/痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记|痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记]]

## Description
百丈怀海是唐代禅宗高僧，被尊称为"百丈禅师"。他在禅宗发展史上具有重要地位，制定了《百丈清规》，为禅宗寺院的组织制度与修行规范奠定了基础。在《[[entities/百喻经|百喻经]]》的讲解中，百丈怀海被引用为修行理念的权威依据。他提出"神通妙用，运水搬柴"的著名理念，主张真正的修行不在于追求玄妙的神通境界，而在于在搬柴运水等日常事务中保持当下的觉察。这一理念强调，修行应落实于日常生活的每一个当下，反对好高骛远、追求直接开悟而忽略持戒、打坐等基础功夫的心态，是破除[[concepts/慢心|慢心]]的重要法义依据，也与[[concepts/正念|正念]]修行的核心理念相契合。
