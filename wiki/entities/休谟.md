# 休谟

## Basic Information
- Type: 人物（Person）
- Source: [[sources/冰与火之歌粉丝升级手册]]

## Description

大卫·休谟（David Hume，1711-1776）是苏格兰哲学家、历史学家，被视为最重要经验主义哲学家之一。他对西方哲学产生了深远影响，特别是在认识论领域提出了著名的"归纳问题"，质疑从过去的经验推论未来知识的合理性。休谟还深入分析了因果关系的本质，认为人类所谓的因果联系不过是习惯性联想的结果，而非客观存在的必然关系。

在讨论"邪恶问题"时，休谟指出，如果没有全能的上帝，逻辑上的邪恶难题就不会产生——这一洞见旨在批评那种认为邪恶存在必然意味着上帝不全能的论证。他还表现出明显的怀疑主义倾向，对人类知识的确定性和可靠性持审慎态度。
