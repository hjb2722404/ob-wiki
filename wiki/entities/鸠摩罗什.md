# 鸠摩罗什

## Basic Information
- Type: person
- Source: [[sources/百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记|百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记]]

## Description
鸠摩罗什（Kumārajīva）是东晋时期最重要的佛经译师之一，生于龟兹，是将印度佛教论典系统翻译介绍到汉地的关键人物。在中观法脉的东传脉络中，鸠摩罗什承接 [[entities/龙树菩萨|龙树菩萨]] 开创、 [[entities/提婆菩萨|提婆菩萨]] 推向顶峰的 [[concepts/中观学派|中观学派]] 思想，将《百论》《中论》《十二门论》三部核心中观论典译为汉文，为中观学说在中土扎根奠定了文本基础。其译著直接启发了后来 [[entities/吉藏|吉藏]]（嘉祥大师）以三论为宗建立 [[entities/三论宗|三论宗]]，并以 [[concepts/破邪显正|破邪显正]] 与 [[concepts/二谛圆融|二谛圆融]] 为宗旨，完成了印度中观法脉到中国佛教宗派的转化，是中观思想"火光相传"千年不断的关键一环。
