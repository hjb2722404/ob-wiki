# 《笑林》

## Basic Information

## Description
《笑林》是中国三国时期由[[entities/邯郸淳|邯郸淳]]编撰的笑话集，被认为是中国最早的笑话集之一。该书广泛吸收了来自[[entities/百喻经|《百喻经》]]的愚人故事素材，是佛教譬喻文学在中国世俗化传播的重要见证。例如，《笑林》收录了改编自《百喻经》"治秃喻"的世俗笑话，但去掉了原本的佛法开示部分，仅保留了笑点。这一改编现象体现了《百喻经》故事从宗教文本向世俗文本转化的传播途径，也展示了[[concepts/应机说法|应机说法]]的精神在跨文化传播中向世俗化方向的演变。
