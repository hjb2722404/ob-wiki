# 曼昆

## Basic Information
- Type: 人物（person）
- Source: [[_raw/读书笔记/经济理财/《曼昆经济学原理》（二）.md]]

## Description
曼昆（N. Gregory Mankiw）是美国著名经济学家，现任哈佛大学经济学教授。他是全球最畅销的经济学教材《经济学原理》（Principles of Economics）的作者，该教材被广泛应用于全球各大高校的本科经济学课程。本页面所阅读的资料来源于曼昆《经济学原理》第二卷微观经济学部分。曼昆教授曾在多届美国政府中担任经济顾问职务，为政策制定提供专业建议。他擅长以通俗易懂的方式阐释经济学基础概念，使初学者能够快速掌握经济学的核心原理。曼昆的经济学教学理念强调将复杂理论简化为生活实例，帮助学生理解日常生活中无处不在的经济学逻辑。
