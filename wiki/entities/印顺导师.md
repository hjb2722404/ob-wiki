# 印顺导师

## Basic Information
- Type: person
- Source: [[sources/阿含经核心教义讲解从原始佛教到大乘人间佛教的完整脉络-2026年06月04日-来自Get-笔记|阿含经核心教义讲解从原始佛教到大乘人间佛教的完整脉络-2026年06月04日-来自Get-笔记]]

## Description
印顺导师是近现代华人佛学界最重要的学者之一，其佛学考证在本文中具有核心地位。他通过严谨的文献考证提出，《阿含经》并非小乘经典，而是三乘共依的圣典，是大乘佛教一切思想的根源，这一观点纠正了汉传佛教千年来对阿含经的误解。他进一步指出，空性思想早已存在于阿含经中，龙树菩萨的中观学说只是对阿含经思想的系统化表达而非创新。他主张，理解大乘必须先从《[[concepts/za-ahan-jing|杂阿含经]]》打好基础。他所提出的人间佛教理念源自阿含经，核心是[[entities/shijiamouni-fo|释迦牟尼佛]]在人间成佛，佛法为人间而说，修行在人间完成，解脱在人间证得，对当代佛教发展影响深远。
