# 埃里卡

## Basic Information
- Type: person
- Source: [[_raw/读书笔记/教育学习/马文·柯林斯的教育之道：通往卓越教育的路径-马文·柯林斯 希维娅·塔玛金.md]]

## Description

埃里卡(Erica)是《[[sources/马文·柯林斯的教育之道通往卓越教育的路径-马文·柯林斯-希维娅·塔玛金|马文·柯林斯的教育之道]]》一书中重点记录的一个曾被贴上"问题学生"标签的女孩。她的转变过程是马文·柯林斯教育方法的典型例证。在书中，柯林斯描述了埃里卡如何观察、试探老师的可信度和态度，并通过持续的赞扬、拥抱和无条件的接纳，最终使埃里卡决定安定下来，开始认真学习。柯林斯认为孩子的改变最终来自于他们自己的决定，而不是外在的威胁，这印证了书中关于自我接纳和自发改变的教育理念。
