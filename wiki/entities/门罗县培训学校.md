# 门罗县培训学校

## Basic Information
- Type: 组织（教师培训机构）
- Source: [[sources/马文·柯林斯的教育之道通往卓越教育的路径-马文·柯林斯-希维娅·塔玛金|马文·柯林斯的教育之道通往卓越教育的路径]]

## Description
门罗县培训学校是马文·柯林斯（Marva Collins）职业生涯早期接受教师培训的重要场所。在这所培训机构中，柯林斯遇到了一位严格的校长，这位校长对新进教师有着极高的要求。据柯林斯在书中描述，这位校长会连续两个月每天坐在教室里观察新教师的课堂表现，一边摇头或点头，一边详细记录他们的教学行为。下课后，校长会像对待自己孩子一样给予新教师一对一的上课和指导。这位校长训练柯林斯学会观察学生的面部表情，通过眼睛判断学生是否理解教学内容。校长强调每节课都要直奔主题，关注每一个学生的参与情况，特别是那些落后于课堂进度的学生。这段经历让柯林斯深刻理解了优秀教师不仅需要掌握所教内容，还需要深入了解每一个学生，成为她后来教育实践的重要基础。
