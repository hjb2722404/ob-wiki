# 忧郁的艾迪

## Basic Information
- Type: person
- Source: [[sources/冰与火之歌]]

## Description
忧郁的艾迪是守夜人军团（Night's Watch）的一员，以其独特的黑色幽默和忧郁气质著称。他是琼恩·雪诺的重要伙伴和朋友，以在黑暗中提着灯笼引导琼恩的经典场景而闻名。他的幽默带有自嘲色彩，常常用幽默来掩饰生活的艰辛。作为守夜人军团中最具喜剧色彩的角色之一，艾迪的存在为这部宏大的史诗作品增添了一抹独特的黑色幽默。他的经典台词充满了对生活无奈的调侃，体现了北境守夜人在漫长黑夜中的生存智慧。

- "小心老鼠，大人。"忧郁的艾迪引领琼恩走下台阶，手里提着一盏灯笼。"如果你踩到它们，它们会发出可怕的尖叫。我小时候，我妈妈经常发出同样的声音。她一定有不少老鼠的血统，现在我想起来了。棕色的头发，明亮的小眼睛，喜欢奶酪。也许她也有一条尾巴，我从没留神去看。"
