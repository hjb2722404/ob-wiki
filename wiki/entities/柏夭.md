# 柏夭

## Basic Information
- Type: 人物
- Source: [[Clippings/最早来到山丹的两位古代帝王.md]]

## Description
柏夭是[[entities/周穆王|周穆王]]西巡时期河西走廊地区的部落首领。据史料记载，周穆王从洛阳出发西巡，经河套进入河西走廊，经过删丹后继续西行，过黑河抵达[[concepts/居延海|居延海]]。在当地部落首领柏夭的向导下，周穆王一行一直抵达"羽琌之地"。柏夭作为向导，在周穆王西游行程中发挥了重要的引领作用，体现了周朝与西北游牧民族之间的友好交往关系。柏夭的身份表明，当时河西走廊地区已存在有组织的部落势力，能够为远道而来的中原王朝使团提供向导服务。
