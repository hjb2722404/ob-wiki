# 琼恩

## Basic Information
- Type: person
- Source: [[sources/冰与火之歌]]

## Description
琼恩·雪诺是乔治·R·R·马丁所著的《冰与火之歌》系列中的核心虚构角色之一。他是临冬城公爵艾德·史塔克的私生子，在故事初期作为守夜人军团的新成员登场。琼恩在成长过程中经历了来自各方的歧视与挑战，从被轻视的私生子逐渐成长为北境的关键领袖人物，最终成为守夜人军团的总司令，展现了坚韧不拔的意志和面对困境时的决心。他的故事线贯穿整个系列，体现了身份认同、荣誉与牺牲等主题，与《权力的游戏》的核心叙事紧密相连。
