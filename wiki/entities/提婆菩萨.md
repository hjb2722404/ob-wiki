# 提婆菩萨

## Basic Information
- Type: person
- Source: [[sources/百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记|百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记]]

## Description

提婆菩萨又名迦那提婆（Āryadeva），是[[entities/龙树菩萨|龙树菩萨]]最著名的弟子，[[concepts/中观学派|中观学派]]第二代宗师。佛陀涅槃八百年后，外道思想泛滥，各派执着存在不变实体，提婆造[[entities/百论|《百论》]]以防护正见、遮止邪见。提婆曾闯入大自在天神殿取出神像眼睛，以两难逻辑破斥[[concepts/神我论|外道神我论]]，其论辩风格犀利而严密。提婆留下三部中观论典：100颂的《百论》、400颂的《四百论》、100字的《百字论》。他在波吒厘城论辩击败所有外道后，被怨恨的外道弟子刺杀，临终留下"诸法本空"的教诲，以死亡示现无我证量。其著作经[[entities/鸠摩罗什|鸠摩罗什]]译介入中土，并由[[entities/吉藏|吉藏]]等中国佛教学者深入阐发，对汉传佛教三论宗影响深远。
