# 长阿含经

## Basic Information
- Type: other
- Source: [[sources/阿含经核心教义讲解从原始佛教到大乘人间佛教的完整脉络-2026年06月04日-来自Get-笔记|阿含经核心教义讲解从原始佛教到大乘人间佛教的完整脉络-2026年06月04日-来自Get-笔记]]

## Description
长阿含经是汉传佛教阿含经四部之一，收录的都是篇幅较长的经典。该经内容多为[[entities/释迦牟尼佛|释迦牟尼佛]]与外道辩论、宇宙观阐述、社会伦理教导，包含大量世间法内容。值得注意的是，部分天界鬼神描述是随顺当时印度文化的方便说法，其目的是帮助读者理解佛陀对世间生活的教导。

长阿含经与[[entities/杂阿含经|杂阿含经]]、[[entities/中阿含经|中阿含经]]、[[entities/增一阿含经|增一阿含经]]共同构成汉传佛教阿含经的整体，对应南传佛教的五部尼柯耶（Dīrghāgama 在南传佛教中归入《长部》）。在[[concepts/人间佛教|人间佛教]]的视角下，长阿含经因其丰富的世间法内容与社会伦理教导，被视为佛教关注现实社会生活、回应世间问题的重要经典依据。
