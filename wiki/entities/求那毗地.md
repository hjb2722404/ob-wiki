# 求那毗地

## Basic Information
- Type: person
- Source: [[sources/痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记|痴华鬘千年佛教典籍百喻经的智慧与现代启示-2026年06月04日-来自Get-笔记]]

## Description
求那毗地（梵名 Guṇabhadra）是来自天竺的高僧，活跃于南朝齐梁之际。公元492年（南齐永明年间），他在[[entities/建康|建康]][[entities/正观寺|正观寺]]完成了[[entities/百喻经|《百喻经》]]的汉文翻译工作。他在序言中提出，故事以"笑话"包裹"良药"，读者需透过笑声领悟真义，体现了"应机说法"的弘法智慧。其译本的语言采用介于文言和白话之间的中间体，大量使用四言句式，符合汉语的音节节奏，读来朗朗上口、便于记忆，是他有意为之的本土化选择——这种"文字般若"的运用方式使佛理得以深入中土文化。译文中还保留了"由旬""欢喜丸""阎浮提"等梵语音译词汇，是中印文明交汇的珍贵文学遗存。其翻译工作直接承继了《百喻经》原典的编纂者[[entities/僧伽斯那|僧伽斯那]]所整理的譬喻素材。
