# 加里

## Basic Information

- Type: person
- Source: [[sources/马文·柯林斯的教育之道通往卓越教育的路径-马文·柯林斯-希维娅·塔玛金]]

## Description

加里（Gary）是书中记录的一个有抵触情绪的学生，曾经只完成作业的一半。书中详细描述了[[entities/马文·柯林斯|马文·柯林斯]]如何通过启发式提问而非强迫来引导他："虽然这个男孩必须改变他生活的重点，但马文并没打算强迫他。总有一天，在无数次赞扬和拥抱之后，他的抵触情绪会自动消解。"加里的案例展示了柯林斯耐心等待学生自然转变的教育智慧，相信当学生感到被接纳和重视时，他们会主动选择改变。在书中，加里与[[entities/特雷西|特雷西]]、[[entities/埃里卡|埃里卡]]同为柯林斯教育方法下的学生典型，代表了[[concepts/积极强化法|积极强化法]]在教育实践中的应用。
