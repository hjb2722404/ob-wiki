# 杂阿含经

## Basic Information
- Type: other
- Source: [[sources/阿含经核心教义讲解从原始佛教到大乘人间佛教的完整脉络-2026年06月04日-来自Get-笔记|阿含经核心教义讲解从原始佛教到大乘人间佛教的完整脉络-2026年06月04日-来自Get-笔记]]

## Description
杂阿含经（梵文：Saṃyuktāgama），又称相应阿含，是四部阿含经中最古老、最根本的一部经典。"杂"字取杂糅汇集之意，该经按照九分教体例汇编，以蕴、处、缘起、圣谛等修行次第为纲要，文字精炼，直指修行核心，是禅修者公认的修行指南。

[[entities/印顺导师|印顺导师]]曾明确提出，要理解大乘佛教必须先从杂阿含经打好基础。杂阿含经中已包含第一义空经等空性思想，是大乘空性思想的源头之一。文中进一步指出，龙树菩萨的中观思想并非凭空创新，而是以更系统的方式重新表达了阿含经中的根本道理，属于"回归"而非"创立"。杂阿含经因此成为论证阿含经为一切佛法根源的重要经典依据。
