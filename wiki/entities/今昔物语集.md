# 《今昔物语集》

## Description
《今昔物语集》（今昔物語集，Konjaku Monogatarishū）是日本平安时代末期的佛教说话文学集，成书于12世纪左右，全书共三十一卷，收录故事千余则。该书在内容上广泛吸收了源自印度的佛教故事，包括[[entities/百喻经|《百喻经》]]中的寓言譬喻，将其与日本本土的世俗故事、历史传说和神灵怪异谭相融合。作为佛教文学东传的重要一环，《今昔物语集》将印度佛教寓言与日本本土文化相结合，"结合本土文化开出新的文学之花"，体现了佛教传播过程中在地化的典型路径。它既是[[entities/百喻经|《百喻经》]]东亚传播的重要见证，也是中日文学与佛教文化交流的经典例证，对后世日本文学（如《宇治拾遗物语》《徒然草》等）产生了深远影响。
