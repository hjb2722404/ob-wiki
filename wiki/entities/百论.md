# 《百论》

## Basic Information

## Description
《百论》（梵文：Śataśāstra）是[[entities/提婆菩萨|提婆菩萨]]所造的中观论典，全书共有十品，每品十首偈颂，合计一百颂。该论以破邪显正为主旨，从捨罪福开始逐步剥开众生对实体的执着，结构清晰、逻辑严密，被视为最适合作为[[concepts/中观学派|中观]]入门的经典。提婆留下与《百论》相关的三部著作，分别是100颂的《百论》、400颂的《四百论》、100字的《百字论》。

汉传《百论》由[[entities/鸠摩罗什|鸠摩罗什]]大师翻译，后世[[entities/吉藏|吉藏]]以《百论》连同《中论》《十二门论》三论为宗建立[[entities/三论宗|三论宗]]。全论十品依次为：捨罪福、破神、破一、破异、破情、破尘、破因中有果、破因中无果、破常、破空，层层递进，每一品都是一层剥落、一次松绑，从放下对福德的执着开始，逐步拆解永恒自我、整体与部分、感知与物质、因果两端、永恒实体等边见，最终拆解对空的执着，指向离戏论的实相，契入[[concepts/缘起性空|缘起性空]]的甚深法义。
