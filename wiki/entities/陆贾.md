# 陆贾

## Basic Information
- Type: person
- Source: [[Clippings/最早来到山丹的两位古代帝王.md]]

## Description
陆贾是西汉著名文学家，其学术成就主要体现在他所著的《新语》一书中。《新语》是西汉初期的重要政论著作，内容涵盖君臣治国之道及历史经验总结。陆贾在《新语》中明确记载"大禹出于西羌"，这一文献记载为后世研究大禹族源提供了重要依据，表明大禹与西部地区的密切联系。作为西汉初年的文官，陆贾兼具政治家与学者双重身份，其思想对后世有一定影响。
