# 林则徐

## Basic Information
- Type: 人物
- Source: [[Clippings/最早来到山丹的两位古代帝王.md]]

## Description
林则徐（1785年－1850年）是清朝著名政治家，以禁烟闻名于世，被誉为"民族英雄"。1842年，因虎门销烟事件触怒英国而被道光帝贬赴新疆伊犁戍边。在其戌边途中的著作《荷戈纪程》里，林则徐详细记载了他途经山丹城西渡河时亲眼所见《大禹导弱水碑》的经历。这块碑即大名鼎鼎的夏禹碑，亦称禹碑、岣嵝碑，最早见于东汉文献，相传为夏禹治水所立，原迹湮没。现存最早版本为南宋时发现于湖南衡山岣嵝峰，宋明以降各地多有摹刻本。碑文为蝌蚪文，明代杨慎曾译为汉文77字。林则徐的记录为这一历史文物提供了清代存在的确凿证据，证明该碑至少在清末仍存在于山丹，具有重要的文物史证价值。
