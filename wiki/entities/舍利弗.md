# 舍利弗

## Description
舍利弗（梵文：Śāriputra）是佛陀座下智慧第一的弟子，是阿含经中记载的重要佛弟子代表之一。据经典记载，他听闻阿说示比丘宣说"诸法因缘生，诸法因缘灭"之偈语后，便当下证得预流果，展现了极高的悟性与对[[concepts/缘起法|缘起法]]的深刻理解。舍利弗擅长解说义理，是佛陀说法时最重要的侍者与对谈者之一。他性格极其谦卑包容，曾因被年轻比丘无端指责而当众顶礼谢罪，成为佛教中谦卑典范的代表。舍利弗在[[entities/释迦牟尼佛|释迦牟尼佛]]之前入灭，与[[entities/阿难|阿难]]（多闻第一）共同构成佛陀弟子中"智慧"与"多闻"的互补象征。
