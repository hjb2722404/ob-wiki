# 特雷西

## Basic Information
- Type: person
- Source: [[sources/马文·柯林斯的教育之道通往卓越教育的路径-马文·柯林斯-希维娅·塔玛金|马文·柯林斯的教育之道通往卓越教育的路径]]

## Description

特雷西（Tracy）是[[sources/马文·柯林斯的教育之道通往卓越教育的路径-马文·柯林斯-希维娅·塔玛金|马文·柯林斯]]所著教育案例中的一个典型学生形象，她是德拉诺小学的学生。由于性格安静内敛，特雷西在学校中经常被老师忽视，无论在哪间教室都被安排坐在最后一排。这个案例深刻揭示了学校体制如何系统性地边缘化某些学生——不是因为他们能力不足，而是因为他们不符合教师对"活跃学生"的期待。当[[entities/马文·柯林斯|马文·柯林斯]]遇到特雷西时，她没有让这个孩子坐到最后一排，而是将她带到前排靠近同学辛迪旁边的座位上。特雷西代表了那些被学校体制忽视但实际上具有巨大潜力的学生，她的故事成为了[[concepts/拒绝标签化|拒绝标签化]]这一教育理念的有力例证。
