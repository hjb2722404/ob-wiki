# 四百论

## Basic Information
- Type: product
- Source: [[sources/百论全品深度解读提婆菩萨的中观破执智慧与当代应用-2026年06月04日-来自Get-笔记|百论全品深度解读：提婆菩萨的中观破执智慧与当代应用]]

## Description
《四百论》（梵文：Catuhśataka）是[[entities/提婆菩萨|提婆菩萨]]所著的中观论典之一，全文共有400颂（偈颂），是提婆三部中观论著中篇幅最大的一部。与100颂的[[entities/百论|《百论》]]和100字的[[entities/百字论|《百字论》]]相比，《四百论》结构更为宏大，更全面地展开了提婆的中观思想。该论著与《百论》《百字论》共同构成了[[concepts/中观学派|中观学派]]破斥外道、阐扬空性的重要文本。在提婆的三部论著中，《百论》因结构清晰、逻辑严密，被认为最适合作为中观入门的读本，而《四百论》则以更广博的篇幅深入展开中观破执的义理。
