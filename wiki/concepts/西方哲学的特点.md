# 西方哲学的特点

[[邓晓芒]]论西方哲学的核心特点——**"爱智慧"（Philosophy）vs "智慧之学"**。 ^[extracted]

## 核心命题："爱智慧"≠"智慧之学"

[[亚里士多德]]总结：哲学起源于"**惊异**"——一种超功利的兴趣，纯粹的"爱"的追求。 ^[extracted]

| | 中国哲学 | 西方哲学（古希腊） |
|--|---------|-----------------|
| 目的 | 经世致用（治国平天下、养生、人际和谐） | **对智慧本身的热爱**（超越一切目的） |
| 态度 | 计较功利，追求实用 | **不计功利，只求兴趣** |
| "哲学"的理解 | 智慧之学（"道可道"） | 爱智慧（Philosophy = 对智慧的爱） |
| 代表人物 | 于丹（讲《论语》《庄子》） | 泰勒斯（观察星空掉入水坑） |

## 泰勒斯的故事

泰勒斯为观察星空而失足掉入水坑，遭到女奴嘲笑。但他一旦把天象知识用于预测橄榄油丰收，就发了大财——这说明古希腊哲人**知道知识的用处，却不看重这种用处**。 ^[extracted]

## 柏拉图与亚里士多德：超越性

- 柏拉图划分感性物质世界 vs 超越的精神世界 → 西方的"唯心主义"传统
- 亚里士多德称"第一哲学"为"**神学**"——哲学具有神圣的超越性

> 哲学不是用于别的目的，而是**超越于一切目的**。 ^[extracted]

## 邓晓芒对"中国没有哲学"的判断

[[entities/雅克·德里达|雅克·德里达]]说中国没有"哲学"——邓晓芒认为这里有**文化错位**：

- 如果"哲学" = **爱智慧**（Philosophy）→ 中国从来没有过，今天也没有
- 如果"哲学" = **智慧之学**（工具性智慧）→ 中国当然有 ^[extracted]

## 中国文化的缺失

[[鲁迅]]把中国社会形容为"**无爱的人间**"——中国文化缺乏对纯粹精神性的追求，只关心肉体（或与肉体不分的精神）。邓晓芒认为中国文化需要补上"爱智慧"这一课。 ^[extracted]
